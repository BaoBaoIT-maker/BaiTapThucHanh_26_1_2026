# Dự án Landing Page

## Giới thiệu
Đây là dự án thực hành môn Công cụ và Môi trường phát triển phần mềm. Mục tiêu của dự án là xây dựng một trang Landing Page cơ bản và làm quen với quy trình làm việc nhóm trên GitHub thông qua mô hình Gitflow.

## Cấu trúc dự án
Dự án được tổ chức theo cấu trúc lắp ghép các thành phần HTML:
- `index.html`: File chính điều hướng và nạp các section.
- `style.css`: File định dạng giao diện chung cho toàn trang.
- `sections/`: Thư mục chứa các tệp HTML con cho từng phần của trang web.

## Công nghệ sử dụng
- Ngôn ngữ: HTML5, CSS3, JavaScript (Fetch API).
- Công cụ: VS Code, Live Server extension.
- Quy trình: Gitflow (main, develop, feature/*).

## Cách chạy dự án
1. Clone repository về máy local.
2. Mở thư mục dự án bằng VS Code.
3. Chuột phải vào file `index.html` và chọn **Open with Live Server**.
4. Truy cập địa chỉ `http://127.0.0.1:5500` trên trình duyệt.

## Giấy phép
Dự án này sử dụng giấy phép MIT.
