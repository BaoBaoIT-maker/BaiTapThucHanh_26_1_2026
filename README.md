# Dự án Landing Page

## Giới thiệu
[cite_start]Đây là dự án thực hành môn Công cụ và Môi trường phát triển phần mềm[cite: 1]. [cite_start]Mục tiêu của dự án là xây dựng một trang Landing Page cơ bản và làm quen với quy trình làm việc nhóm trên GitHub thông qua mô hình Gitflow[cite: 3, 4].

## Cấu trúc dự án
[cite_start]Dự án được tổ chức theo cấu trúc lắp ghép các thành phần HTML[cite: 13]:
- [cite_start]`index.html`: File chính điều hướng và nạp các section[cite: 14].
- [cite_start]`style.css`: File định dạng giao diện chung cho toàn trang[cite: 15].
- [cite_start]`sections/`: Thư mục chứa các tệp HTML con cho từng phần của trang web[cite: 16].

## Công nghệ sử dụng
- [cite_start]Ngôn ngữ: HTML5, CSS3, JavaScript (Fetch API)[cite: 22, 35].
- [cite_start]Công cụ: VS Code, Live Server extension[cite: 49].
- [cite_start]Quy trình: Gitflow (main, develop, feature/*)[cite: 71, 74].

## Cách chạy dự án
1. Clone repository về máy local.
2. Mở thư mục dự án bằng VS Code.
3. Chuột phải vào file `index.html` và chọn **Open with Live Server**[cite: 51].
4. Truy cập địa chỉ `http://127.0.0.1:5500` trên trình duyệt[cite: 51].

## Giấy phép
Dự án này sử dụng giấy phép MIT[cite: 65].
